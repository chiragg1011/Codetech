# CODETECH Internship Tasks

This repository contains all internship tasks completed during the CODETECH Full Stack Development Internship Program using modern web technologies and development tools.

---

## 📁 Tasks Included

### 🚀 Task 1 - SkyFlow (API Integration Web Application)

A premium, glassmorphic weather forecasting dashboard that fetches and displays real-time meteorological metrics (such as temperature, UV index, wind speed, sunset, and sunrise) using the Open-Meteo REST API.
* **Features:** Dynamic auto-geolocations, responsive metric-to-imperial toggles, light/dark modes, and autocomplete city searches.
* **Folder:** [`/task1`](file:///C:/Users/Dell/.gemini/antigravity/scratch/Codetech/task1)

### 💬 Task 2 - AeroChat (Real-Time Chat Application)

A modern, high-fidelity real-time workspace designed for instant collaborative messaging utilizing bi-directional Socket.IO channels.
* **Features:** Custom color accent settings for active users, mobile responsive sliding drawer for online users, live typing status beacons, and system notifications.
* **Folder:** [`/task2`](file:///C:/Users/Dell/.gemini/antigravity/scratch/Codetech/task2)

### 📝 Task 3 - SyncQuill (Real-Time Collaborative Document Editor)

A state-of-the-art document co-authoring workspace modeled after Google Docs, utilizing WebSocket increments for concurrent multi-user editing.
* **Features:** Character-level delta synchronizations, active collaborator avatar stacks, debounced cloud auto-saving to MongoDB (with high-end local JSON database offline fallback), document template builders, and direct hash-based room link sharing.
* **Folder:** [`/task3`](file:///C:/Users/Dell/.gemini/antigravity/scratch/Codetech/task3)

### 🧩 Task 4 - FocusFlow (Chrome Extension for Productivity Analytics)

An advanced SaaS-style productivity tracking assistant similar to RescueTime and ActivityWatch, designed to monitor and optimize time spent online.
* **Features:** Manifest V3 chrome active tab time extraction, Chrome Idle API auto-inactivity pausing, motivational glassmorphic blocker redirection screens, synchronized Pomodoro timer popups, weekly reports (including category donut charts and weekly bar distributions), and heuristics-based AI productivity suggestion alerts.
* **Folder:** [`/task4`](file:///C:/Users/Dell/.gemini/antigravity/scratch/Codetech/task4)

* **Frontend Frameworks & Engines:** React.js, Vite, Quill.js, Recharts, HTML5, CSS3, Tailwind CSS
