Node.js Server Required

AeroChat operates using real-time WebSockets which requires a running backend server. It looks like you've opened the HTML file directly in the browser using the file:// protocol, or the backend server is not running.

How to run the application correctly:

  1. Ensure Node.js is installed on your computer.
  2. Open your terminal and run npm install in the project folder.
  3. Start the server by running npm start or npm run dev.
  4. Open http://localhost:3002 in your browser to chat!
